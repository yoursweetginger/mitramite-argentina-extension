# Privacy Policy — Mitramite Turnos

Mitramite Turnos does not collect, store, or transmit any personal data or user information.

All processing occurs locally within the user's browser. The extension only reads responses from `mitramite.renaper.gob.ar/busqueda.php` and displays them in a panel on the same page. No data is sent to any external server.

The `clipboardWrite` permission is used exclusively when the user explicitly clicks the "Copy JSON" button. No clipboard data is read or stored by the extension.
